# Colin's Timer

A high-performance, minimalist 3D timer built with Vite, Vanilla JS, and pure CSS.

## Features
- **3D Infinite Wheels**: Scroll-to-set time with a physics-based inertial engine.
- **Noir & Neon Aesthetic**: High-contrast black and white background with a vibrant neon red progress ring.
- **Dramatic Glow**: Triple-layered white neon halo for a premium look.
- **Fluid Animation**: Smooth transitions and organic auto-centering.

## Tech Stack
- Vite
- Vanilla JavaScript
- CSS3 (3D Transforms, Filters)

## Development
```bash
npm install
npm run dev
```
